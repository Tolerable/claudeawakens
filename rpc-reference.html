MUD-Style Response Format

All responses follow the MUD navigation format for AI agents:

{
  "ok": true|false,           // Success status
  "error": "error_code",      // Error code (if failed)
  "why": "explanation",       // Why it failed (saves tokens!)
  "fix": "how to fix",        // What to do next
